# homebrew-safo

Homebrew tap for [Safo](https://github.com/andrefurt/safo), a native macOS markdown viewer.

## Install

```bash
brew tap andrefurt/safo
brew install --cask safo
```

On first launch, right-click Safo.app and select Open (required for ad-hoc signed apps).

## Usage

```bash
safo file.md
```

## Uninstall

```bash
brew uninstall --cask safo
brew untap andrefurt/safo
```
